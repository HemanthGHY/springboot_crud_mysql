# Student Management System - Spring Boot CRUD (MySQL)

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- REST API

## Features
- Create Student
- Update Student
- Delete Student
- Get Student by ID
- Get Student by NAME
- Get Student by EMAIL
- Get All Students

## How to Run
1. Create MySQL DB: student_db
2. Update application.properties
3. Run Spring Boot Application
4. Test APIs using Postman
